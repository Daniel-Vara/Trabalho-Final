HTML - Estrutura da Web

Exemplo de estrutura HTML básica

O que é HTML?

HTML é a linguagem de marcação padrão para criar páginas web. Ele define a estrutura e o significado do conteúdo através de elementos e tags.

HTML não é uma linguagem de programação, mas sim de marcação que estrutura o conteúdo da web.

Elementos Essenciais

Estrutura Básica

<!DOCTYPE html> <html> <head>...</head> <body>...</body> </html>

Cabeçalhos

<h1>Título</h1> <h2>Subtítulo</h2> <!-- Até h6 -->

Links

<a href="url">Texto</a>

Imagens

<img src="imagem.jpg" alt="Texto">

Listas

<ul> <li>Item</li> </ul>

Divisões

<div class="classe">...</div>

CSS - Estilização da Web

O Que é CSS?

CSS é a linguagem que descreve a apresentação de documentos HTML, controlando layout, cores, fontes e outros aspectos visuais.

CSS permite separar completamente a estrutura (HTML) da apresentação (estilos), facilitando a manutenção.

Conceitos Fundamentais

Seletores

Padrões que selecionam elementos para estilização

p { } /* Todos os parágrafos */ .classe { } /* Por classe */ #id { } /* Por ID */

Box Model

Modelo que define margem, borda, preenchimento e conteúdo

margin: 10px; padding: 20px; border: 1px solid;

Flexbox

Modelo de layout unidimensional para design responsivo

display: flex; justify-content: center; align-items: center;

Grid

Sistema de layout bidimensional para designs complexos

display: grid; grid-template-columns: 1fr 1fr;

Media Queries

Adapta o layout para diferentes tamanhos de tela

@media (max-width: 768px) { }

Animações

Cria transições e animações sem JavaScript

transition: all 0.3s; @keyframes anim { }
