O que é GitHub?

O GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele permite que desenvolvedores colaborem em projetos de qualquer lugar do mundo.

GitHub é mais que uma ferramenta - é a rede social dos desenvolvedores e o maior repositório de código aberto do mundo.

- Comunidade Dev

Principais benefícios:

  • Controle de versão eficiente
  • Colaboração em equipe
  • Portfólio de projetos
  • Integração contínua (CI/CD)
  • Comunidade open-source

Comandos Git Essenciais

git init

Inicializa um novo repositório Git em um diretório local.

git init

git clone

Clona um repositório existente para sua máquina local.

git clone https://github.com/usuario/repositorio.git

git add

Adiciona alterações ao staging area para o próximo commit.

git add . # Todos os arquivos
git add arquivo.js # Arquivo específico

git commit

Registra as alterações no histórico do repositório.

git commit -m "mensagem descritiva"

git push

Envia commits locais para o repositório remoto.

git push origin main

git pull

Obtém as alterações mais recentes do repositório remoto.

git pull origin main

git merge

Combina alterações de diferentes branches.

git merge nome-da-branch

git log

Exibe o histórico de commits do repositório.

git log --oneline --graph

Fluxo de trabalho comum com Git/GitHub

Fluxo de Trabalho Básico

Um fluxo de trabalho típico no GitHub envolve:

1

Criar um branch

Trabalhe em uma nova funcionalidade ou correção em um branch separado.

2

Fazer commits

Salve suas alterações com mensagens descritivas.

3

Enviar para o GitHub

Use push para enviar seu branch para o repositório remoto.

4

Abrir Pull Request

Solicite a revisão e merge de suas alterações.

5

Revisão e Merge

Após aprovação, as alterações são incorporadas ao código principal.

Recursos do GitHub

Branches

Linhas de desenvolvimento isoladas para trabalhar em features ou correções sem afetar o código principal.

Pull Requests

Solicitações para incorporar alterações de um branch em outro, com revisão de código.

Issues

Sistema de rastreamento de bugs, melhorias e tarefas do projeto.

GitHub Actions

Automatize testes, builds e deploys com CI/CD integrado.

GitHub Pages

Hospedagem gratuita para sites estáticos diretamente do repositório.

Colaboração

Controle de acesso granular, revisão de código e discussões.

O GitHub oferece muitos outros recursos como Projects, Wiki, Security alerts e muito mais. Explore a documentação oficial para descobrir tudo!
